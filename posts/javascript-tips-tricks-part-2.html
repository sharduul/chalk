1. Constructor like thingy in JavaScript

Developers coming from class-based languages are too much used to implementing constructors for initializing their class members. But, we don’t have constructors in JavaScript.

What you can do instead, is use IIFE pattern to initialize your members when the object is first created.

Although, you don’t have to use the IIFE pattern if you don’t want to. Because, in JavaScript, when you say new Foo() it executes all the expressions in the function Foo. But, adding IIFE to your code makes it more structured and is more readable.
